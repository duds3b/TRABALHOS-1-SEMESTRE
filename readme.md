<------ O QUE É O JOGO? ------->
O jogo NEON RESCUE é um jogo feito em p5.js (versão 1.4.2), onde o jogador deve fugir do perseguidor, coletar orbes, fazer a maior pontuação possível, e tentar quebrar o próprio recorde em sua gameplay

<------COMO EXECUTAR--------->
1. Baixe todos os arquivos do projeto

Certifique-se de que estão na mesma pasta:
index.html
sketch.js
musica.mp3
score.mp3
jogador.png
inimigo.png


2. (IMPORTANTE) Use o p5js na versão 1.4.2, o jogo foi 100% desenvolvido nessa versão

Use exatamente estas linhas no seu index.html:
<script src="https://cdnjs.cloudflare.com/ajax/libs/p5.js/1.4.2/p5.min.js"></script>
<script src="https://cdnjs.cloudflare.com/ajax/libs/p5.js/1.4.2/addons/p5.sound.min.js"></script>


2,1. RODE O JOGO USANDO O EDITOR DO P5JS

1 - Entre em: https://editor.p5js.org/

2 - Clique em Upload Folder

3 - Envie a pasta do jogo inteira

4 - Aperte Run


<--------CONTROLES-------->

W A S D → mover

Clique esquerdo do mouse para interagir com os botões
